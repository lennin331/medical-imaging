## Open Source Medical Imaging Data

###Tables
| Column Name       | Description                             |
| ----------------- | --------------------------------------- |
| `image_id`        | Unique identifier for the image         |
| `patient_id`      | Anonymized patient identifier           |
| `study_id`        | Imaging study identifier                |
| `modality`        | Imaging modality (CT, MRI, X-ray, etc.) |
| `body_region`     | Anatomical region scanned               |
| `image_path`      | Relative or absolute path to image file |
| `image_format`    | File format (DICOM, NIfTI, PNG, etc.)   |
| `slice_thickness` | Slice thickness (if applicable)         |
| `resolution`      | Image resolution or voxel spacing       |

###Intent 
- Medical image segmentation research.
- Machine learning model training and evaluation.
- Educational and academic purposes.
